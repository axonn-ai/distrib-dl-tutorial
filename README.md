## Basics of Model Training

### Using PyTorch

```bash
cd session_1_basics/
sbatch --reservation=isc2024 run.sh
```

### Mixed Precision

```bash
MIXED_PRECISION=true sbatch --reservation=isc2024 run.sh
```

### Activation Checkpointing

```bash
CHECKPOINT_ACTIVATIONS=true sbatch --reservation=isc2024 run.sh
```
